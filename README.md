# 🧠 CFO-OS · AI-Native Expense Analyzer

An **AI-powered financial operating system** designed to track, analyze, and optimize expenses with intelligent insights, anomaly detection, and forecasting.

---

## 🚀 Features

### 📊 Smart Dashboard
- Real-time overview of total spending, budget remaining, and transactions
- Interactive charts (category-wise & breakdown)
- Recent transactions & budget usage visualization

### 🤖 AI Agents System
- **Expense Agent** → Auto-categorizes transactions  
- **Anomaly Agent** → Detects unusual spending patterns  
- **Forecast Agent** → Predicts future expenses  
- **CFO Agent** → Generates financial insights  

### 📄 AI CFO Report
- One-click executive financial summary
- Budget health analysis
- Risk detection & cost-saving suggestions

### 💡 AI Insights
- Spending patterns & trends
- Category-wise analysis
- Smart cost-saving recommendations

### 🔔 Smart Alerts
- Budget exceed warnings
- Category-level alerts
- AI-generated actions

### 🎯 Budget Management
- Set budgets (Daily / Monthly / Yearly)
- Auto conversion to monthly
- Category-wise tracking

### 🔮 What-If Scenario Planner
- Simulate cost reductions
- Predict savings & impact instantly

### ➕ Expense Manager
- Add, delete, and manage transactions
- Real-time analysis after entry

### 📊 Demo + Fresh Mode
- *Demo Mode* → Preloaded sample data  
- *Start Fresh* → Clean state  

---

## 🛠️ Tech Stack

- *Frontend:* HTML, CSS, JavaScript  
- *Charts:* Chart.js  
- *Storage:* LocalStorage  
- *Logic:* Rule-based AI + statistical analysis
